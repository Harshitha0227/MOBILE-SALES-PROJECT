Mobile Sales Data Analysis Project-

This project analyzes mobile phone sales data to understand sales trends, customer preferences, and product performance.
Using Python and data analysis techniques, the project explores factors like brand, price, ratings, and sales volume to generate meaningful insights.

___________________________________________________________________________________________________
Tools & Technologies:
	•	Python
	•	Pandas
	•	NumPy
	•	Matplotlib / Seaborn
	•	Jupyter Notebook
_________________________________________________________________________________________________________
Key Features:
	•	Data cleaning and preprocessing
	•	Exploratory Data Analysis (EDA)
	•	Sales trend visualization
	•	Brand-wise and price-wise analysis
	•	Insight generation for business decisions
  __________________________________________________________________________________________________________
Dataset:

The dataset contains information about mobile brands, models, prices, ratings, and sales details.
__________________________________________________________________________________________________________
Objective:

To analyze mobile sales data and identify patterns that help understand market trends and customer buying behavior.
